# feat-arcore

基于 ARCore 的 Android AR 探索模块，当前首个可运行场景是“客厅墙面挂画预览”。

## 当前目标

- 直接进入 AR 预览流
- 扫描客厅中的垂直墙面
- 点击墙面放置一张虚拟挂画
- 切换挂画样式并重新放置
- 为后续真实 3D 画框、贴墙吸附和遮挡效果保留扩展入口

## 当前实现

- 一个可独立打开的 `ArCoreActivity`
- 基于 `ArFragment` 的 AR 相机预览与平面检测
- 仅接受 `Plane.Type.VERTICAL` 的墙面点击放置
- 使用 `ViewRenderable` 把一张 2D 挂画作为 AR 贴画贴到墙上
- 三套内置轻量挂画样式，便于直接在客厅试效果

## 下一步建议

- 换成真实画框图片或本地相册导入
- 增加墙面对齐后的自动吸附与尺寸建议
- 在放置后加入遮挡、阴影和更真实的材质表现
